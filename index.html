Give items a second life

PassItOn is a simple community marketplace where users can list unwanted items, browse available products, and help reduce waste by passing useful items on to others.

How PassItOn Works

1. Add an Item

Users can create a listing with a title, price, description, image and category.

2. Browse Listings

Visitors can view available items and search through different categories.

3. Update Details

Listings can be edited if the price, image, description or category changes.

4. Remove Items

Items can be deleted when they are no longer available.

Why Use PassItOn?

Many useful items are thrown away even though someone else could use them. PassItOn helps people share, sell or give away items locally. The website supports create, read, update and delete functionality using a Supabase database.
